# ChickenFeed
A chicken brush is just a chicken brush

鸡刷，也仅仅是鸡刷

只测试过purpur1.21.11版本，其他的可自行测试。

Only tested with purpur version 1.21.11, others can be tested by yourself.

-------------------------------------------------------------------

无任何权限的普通玩家：右键鸡 → 无掉落，无任何聊天消息。

仅有 chickenfeed.use 权限的玩家：右键鸡 → 正常掉落物品，但无提示消息（因为无 notify 权限）。

拥有 chickenfeed.use 和 chickenfeed.notify 权限的玩家：右键鸡 → 正常掉落物品，并看到成功或冷却提示。

拥有 chickenfeed.reload 权限的 OP：可使用 /chickenfeed reload 重载配置。

Ordinary players with no permissions: Right-click a chicken → no drops, no chat messages.

Players with only the "chickenfeed.use" permission: Right-click a chicken → normal item drops, but no notification messages (because they don't have notify permission).

Players with both "chickenfeed.use" and "chickenfeed.notify" permissions: Right-click a chicken → normal item drops, and see success or cooldown messages.

OPs with chickenfeed.reload permission: can use "/chickenfeed reload" to reload the configuration.
